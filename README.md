# Arsenic Distribution in Nogueiredo Stream (Castrelo de Miño, Ourense)

Interactive map showing arsenic concentrations along the Nogueiredo stream, 
developed as part of my Bachelor's Thesis in Chemistry at the University of Vigo.

## About this project
Arsenic is a naturally occurring contaminant in Galician rivers due to the 
granitic geology of the region. This map visualizes spatial distribution of 
arsenic concentrations along the Nogueiredo stream in Castrelo de Miño (Ourense).

## Built with
- **R** — data analysis and processing
- **Leaflet** — interactive mapping
