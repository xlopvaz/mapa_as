# Environmental Assessment of Nogueiredo Stream (Castrelo de Miño, Ourense)

🗺️ **[View arsenic distribution map](https://xlopvaz.github.io/TFG-Environmental-Assessment-Castrelo/maps/arsenic_map.html)**

🔴 **[View arsenic predictive model](https://xlopvaz.github.io/TFG-Environmental-Assessment-Castrelo/maps/arsenic_predictive_map.html)**

Interactive maps showing arsenic concentrations along the Nogueiredo stream, developed as part of my Bachelor's Thesis in Chemistry at the University of Vigo.

## About this project
Arsenic is a naturally occurring contaminant in Galician rivers due to the metamorphic geology of the region. This project visualizes the spatial distribution of arsenic along the Nogueiredo stream in Castrelo de Miño (Ourense), including a predictive model estimating arsenic concentrations under different precipitation scenarios (wet, normal and dry conditions).

## Built with
- **R** — data analysis, statistics and visualization
- **Leaflet** — interactive mapping
- **ICP-MS-TOF** — multielement analysis
- **MC-ICP-MS** — Sr isotopic ratios
